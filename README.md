# Ndia TVC Roadshow Portal

A professional portal for Ndia Technical and Vocational College roadshow registration verification.

## Features

- **Animated Intro**: Canvas-based bus animation with student waving effect
- **Glass Morphism Design**: Modern glass effect UI with liquid bubbles
- **Responsive Layout**: Works perfectly on all devices (mobile, tablet, desktop)
- **Registration Check**: Search functionality for trainee verification
- **Gallery Section**: Display campus images
- **Course Information**: Detailed course listings
- **Contact Information**: Administration contact details
- **Security Features**: Copy protection, zoom prevention, right-click disable

## Deployment to GitHub Pages

1. Create a new GitHub repository
2. Upload all files to the repository
3. Go to Repository Settings → Pages
4. Select "Deploy from a branch" → "main" branch → "/root" folder
5. Save and wait for deployment
6. Your site will be live at: `https://[username].github.io/[repository-name]`

## Files Structure

- `index.html` - Main HTML file
- `style.css` - All styles including glass effects and animations
- `script.js` - All JavaScript functionality including animations
- `data/trainees.json` - Trainees data (optional, data is already in script.js)
- `assets/` - Logo and other assets (optional, using direct image links)

## Customization

1. **Logo**: Replace the logo URL in HTML line 133
2. **Images**: Update gallery image URLs in HTML lines 297-305
3. **Colors**: Modify CSS variables in `:root` section
4. **Data**: Update trainees data in `script.js` or use external JSON

## Security Features

- Right-click disabled
- Text selection disabled
- Copy/paste/cut disabled
- Zoom prevention
- User scaling restricted
- Context menu disabled

- Chrome 60+
